# Channel GAN: Rayleigh Fading Channel Emulation

This project implements a **Conditional Generative Adversarial Network (CGAN)** to simulate a wireless communication channel. The primary objective is to learn and emulate the conditional probability distribution of a Rayleigh fading channel. This allows the generator to simulate received signals based on transmitted symbols and channel coefficients, without requiring an explicit mathematical model of the Channel State Information (CSI).

## Core Technologies & Architecture
* **Data Source (QuaDRiGa)**: Utilizes realistic radio channel datasets generated by QuaDRiGa, based on the 3GPP 38.901 UMi NLOS scenario (`.mat` format).
* **Model Architecture (WGAN-GP)**: Integrates Wasserstein GAN with Gradient Penalty (WGAN-GP) to ensure training stability and effectively prevent mode collapse.
* **Conditioning Vector**: Both the generator and discriminator are conditioned on the real and imaginary parts of the transmitted signal and the channel coefficients. The formulation is: $[Re(x), Im(x), Re(h), Im(h)]$.

This project is configured and optimized for hardware acceleration on Windows 10/11 systems equipped with an **NVIDIA RTX 4070** GPU. Please follow the steps below to set up your environment:

1. **Create the virtual environment with the specific Python version:**
